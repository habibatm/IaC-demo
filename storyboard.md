# 15-minute presentation storyboard

Core message: DevSecOps is the practical operating model for SSDLC: risk-based controls embedded into developer workflows and CI/CD gates.

1. Why Shift Security Left? - 2 min
Use it to define SSDLC and explain why earlier detection matters.

2. Embedding SSDLC in Existing DevOps and Agile Pipelines - 2 min
Explain that security becomes part of planning, coding, build, merge and deployment routines.

3. SSDLC Lifecycle - 3 min
Do not read every box. Use it as the end-to-end view: design, build, test, deploy, maintain.

4. Security Services Tracks by Application Risk Profile - 3 min
Explain that controls are tailored by risk profile, not one-size-fits-all.

5. Example Toolchain - 2 min
Map practices to tools: planning, code management, automated build, testing, deployment and monitoring.

6. IaC Tool Comparison - 3 min
Use it as the bridge into the demo. Recommend Checkov for the live demo because it is simple to run locally, scans Terraform, and provides clear CLI findings.
